# kasper
programming language in Rust.

## Hello, world.
- Build the program with cargo.
```console
 $ cargo Build
```
- Make source.
```python
write("Hello, Kasper.")
```
- execute the source.
```console
 $ kasper source
 Hello, Kasper.
```

## How to

- Variable declaration:

```python
define name -> T
```
- Variable assignement:

```python
name = value
```
